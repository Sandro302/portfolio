# API для проекта

## Метод импорта проекта OLGA

### Карточка метода

- **HTTP метод:** `POST`
- **Путь:** `/v1/projects/import/olga`
- **Назначение:** Метод используется для импорта инженерного проекта OLGA из набора файлов (.opi, .tpi, .tab)
- **Формат сообщения:** `multipart/form-data`
- **Код успешного ответа:** `201 Created` – Проект успешно импортирован

### Входные параметры

| Параметр | Тип | Формат | Обязательный | Описание |
|----------|-----|--------|--------------|----------|
| opiFile | файл | .opi | Да | Основной XML-файл описания OLGA-проекта |
| tpiFile | файл | .tpi | Нет | Файл шаблона проекта (если не передан — ищется рядом) |
| tabFile | файл | .tab | Нет | Файл состава флюида (если не передан — ищется рядом) |
| projectName | string | [A-Za-z0-9-\\_]+ | Да | Название импортируемого проекта |

### Выходные параметры

| Параметр | Тип | Формат | Обязательный | Описание |
|----------|-----|--------|--------------|----------|
| projectId | string | [a-f0-9\\-]{36} | Да | Уникальный идентификатор проекта (UUID) |
| projectName | string | [A-Za-z0-9-\\_]+ | Да | Название проекта |
